What is the Document Object Model (DOM)?

The document Object Model is a programming language that allows interaction with the webpage's elements. The DOM can be visualized as a upside down tree model where The tree's root is Document. It represent Entire HTML document. Every other elements in the HTML is a node and they are parts of the Document.
A Website use each Nodes or the elements like <h1> <p>as a object datatype.

How do you select an element from the DOM?

We can select an Element from the DOM in several ways.

1. Selecting By ID:
To select an element by its id, we use the document.getElementById() method. For example, document.getElementById("myId").

2. Selecting By Class:
To select elements by their class name, we use the document.getElementsByClassName() method. For example, document.getElementsByClassName("myClass").

3. Selecting By Tag Name:
To select elements by their tag name, we use the document.getElementsByTagName() method. For example, document.getElementsByTagName("p").

4. Selecting By CSS Selector:
To select elements by their CSS selector, we use the document.querySelector() method. For example, document.querySelector("#myId").

5. Selecting By Multiple Elements:
To select multiple elements by their tag name, we use the document.querySelectorAll() method. For example, document.querySelectorAll("p").

What is event delegation in the context of the DOM, and why is it useful?

Event delegation is a technique used in the Document Object Model (DOM) to handle events efficiently. It allows users to attach a single event listener to a parent element instead of attaching a listener to each individual child element. This reduces the number of event listeners and improves performance.

The main advantage of event delegation is that it improves code reusability and reduces memory usage. When an event occurs on the parent element, the event handler function is triggered, and the event object contains information about the specific element that triggered the event.

Event delegation is especially useful when dealing with dynamic content, such as adding or removing elements dynamically. By attaching a single event listener to the parent element, user can handle all the events triggered by the child elements, reducing the number of event listeners and improving efficiency.

How do you manipulate an element's attributes and styles using the DOM?

To manipulate an element's attributes and styles using the DOM, we can use the following methods:

1. Changing Attributes:
To change an element's attribute, we can use the setAttribute() method. For example, element.setAttribute("attributeName", "newAttributeValue").

2. Removing Attributes:
To remove an element's attribute, we can use the removeAttribute() method. For example, element.removeAttribute("attributeName").

3. Changing Styles:
To change an element's style, we can use the style property. For example, element.style.property = "newValue".

4. Accessing Styles:

5. Creating and Appending Elements:
To create a new element, we can use the document.createElement() method. For example, var newElement = document.createElement("div").

To append an element to another element, we can use the appendChild() method. For example, parentElement.appendChild(newElement).

6. Removing Elements:
To remove an element from the DOM, we can use the removeChild() method. For example, parentElement.removeChild(childElement).
